## 📌 프로젝트 개요

본 프로젝트는 산업 환경에서 다수의 AMR(Autonomous Mobile Robot)이  
공유 공간 내에서 이동할 때 발생할 수 있는 충돌을 회피하고,  
경로를 동적으로 재탐색하는 시뮬레이션 프로그램이다.

- 각 로봇은 A* 알고리즘을 이용해 최단 경로를 생성
- 이동 중 충돌 가능 상황이 발생하면 경로를 재탐색
- 우선순위 기반으로 충돌을 회피하며 최종 목표 도달

## 🎯 주요 기능

- ✔ A* 알고리즘 기반 최단 경로 탐색  
- ✔ 다중 로봇(Multi-Agent) 동시 이동

## 🧠 사용 전략

### A*(A-Star) 경로탐색

- g(n): 시작점부터 현재까지 실제 거리  
- h(n): 목표까지의 추정 거리 (맨해튼 거리)  
- f(n) = g(n) + h(n)
👉 다익스트라 알고리즘 + 휴리스틱을 결합한 탐색 방식

### 2. 우선순위 기반 충돌 회피 (Priority-Based Collision Avoidance)
- 다중 로봇 환경에서 각 로봇에 우선순위를 부여
- 동일 경로 또는 충돌 가능 상황 발생 시:
  - 우선순위가 높은 로봇 → 기존 경로 유지
  - 우선순위가 낮은 로봇 → 경로 재탐색 또는 대기

### 3. 동적 재탐색 (Dynamic Replanning)
- 이동 중 충돌 가능 상황이 감지되면 즉시 A* 알고리즘을 재실행
- 변화하는 환경(다른 로봇 위치)을 반영하여 새로운 경로 생성
- 이를 통해 실시간으로 충돌을 회피하며 목표 지점까지 안정적으로 이동
